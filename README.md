# hub-worldcup26-guide

## 项目简介

`hub-worldcup26-guide` 是一个用于归档与发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为页面集合的存储与展示中心，方便对相关主题进行统一浏览与管理。

## 目录说明

```
├── pages/           # 存放所有独立 HTML 页面
├── assets/          # 页面引用的静态资源（图片、样式、脚本等）
├── index.html       # 仓库首页（可选）
└── README.md        # 本文件
```

- `pages/` 目录下按内容或功能组织子目录，每个子目录对应一个独立页面或一组相关页面。
- `assets/` 目录用于存放页面所需的公共资源，避免重复引用。
- 如需新增页面，请遵循现有目录结构，保持组织清晰。

## 页面归档说明

- 所有页面均为独立的 HTML 文件，可直接在浏览器中打开，无需后端服务支持。
- 页面之间不存在跨域依赖，不引用外部不可控资源。
- 归档内容保持原始结构与样式，不进行额外修改或优化。
- 页面可能包含实验性功能或临时内容，仅供参考，不保证长期可用。

## 维护说明

- 本仓库由维护者定期更新，添加新页面或修正已有内容。
- 欢迎提交 Issue 或 Pull Request 提出改进建议。
- 提交新页面时，请确保：
  - 页面为完整 HTML 文件，无外部链接依赖。
  - 文件命名清晰，与内容相关。
  - 将页面放置于合适的子目录下。
- 本仓库不提供任何形式的托管保障，页面内容仅供存档与展示。

## 其他

- 使用本仓库内容时，请遵守相关法律法规与平台政策。
